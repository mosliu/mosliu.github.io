Joplin 真香

摘要

Joplin 真香

当前在用

  1. onenote，配合office 365使用的。 主要使用，挺方便的。手机用也可以，不过不是markdown的，这点就挺烦。支持部分文档加密挺不错的。
  2. joplin，最近用的，挺好用，用onedrive更新。也有clipper的chrome和firefox扩展，主要使用markdown语法，同步速度也不错，挺香的！缺点是没发现支持单页加密的东西，不然可以全面替代onenote了。最主要的是免费啊（office 365 拼车不计了）

选joplin的原因

  1. 数据在onedrive，唯一的风险是被gfw，其他的都好，算是自己掌握数据
  2. 迫穷，免费，开源
  3. MarkDown真舒服
  4. 有clipper，挺好的

弃用的

  1. evernote 慢，之前用过套餐，过期之后，限制几个客户端，好麻烦。弃用。
  2. trello，用于记录信息不是太舒服，偶尔排计划等用
  3. todo 功能单一，没用起来，还不如trello
  4. notion。 绑定的邮箱是gmail，登录时验证麻烦。。最开始的账户被我出了。
  5. wiznote，为知笔记，自搭服务器，算是不错。支持markdown。支持扩展，但是自搭的成本有点高，有技术门槛，还要设定url等，也是个麻烦事。使用官方版还要钱

配套措施

  1. android端打开时加上了密码，省的别人偷看。
  2. Office365 带的onedrive，1T空间，基本上做啥都够了。很方便。同时支持onenote和joplin，我拼车大概一年45RMB，使用正版的office的功能，感觉挺好的
